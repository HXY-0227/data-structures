## 介绍
主要收集了平时练习数据结构和算法的一些代码和笔记。目前主要包括数组、堆、栈、树以及一些leetcode题目和一些算法题目。其中数据结构的部分代码参考自慕课网的课程：玩转数据结构。项目的目录参见下节，其中*.md是介绍对应数据结构的文章，.java文件是代码实现。有些数据只有代码实现没有笔记介绍，有些只有介绍还没有代码实现。后续会补充。
